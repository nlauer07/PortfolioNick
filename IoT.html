IoT Project(s)

Home Automation

This project repurposes a tablet as a dedicated home automation console, effectively replacing standard light switches. A CAD model was designed for mounting the tablet. The server side coordinates with MariaDB for local logging, Python scripts for operational logic, and local LLM voice control for hands-free commands. Sensor automation, WebRTC for real-time streaming, and various APIs enable a seamless smart home experience.
