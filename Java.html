Java Project(s)

SheepScheduler

A console-based program that generates a sheep-shearing schedule using priority rules. Only one sheep can be sheared at a time, and waiting sheep are prioritized by shorter shearing times, then sorted alphabetically.

RobotQueue

Simulates a robot traversing a grid by following a queue of commands. Obstacles and movement logic create a fun programming challenge.

TaskOrganizer

Manages a collection of tasks, each with an action and a priority ranging from 0 (highest) to 5 (lowest). Tasks are distributed into lists according to their priority level.

PrizeShowdownGame

A Java version of a familiar game show's final round. Randomly selects five prizes, and the user guesses the total cost within a target range to win.

VectorMathProgram

Performs vector operations such as addition, subtraction, and calculating magnitude.
